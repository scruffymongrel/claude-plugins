# scruffymongrel/claude-plugins

A [Claude Code](https://code.claude.com) plugin marketplace.

## Installation

Inside Claude Code:

```text
/plugin marketplace add scruffymongrel/claude-plugins
/plugin install <plugin>@scruffymongrel
```

## Plugins

| Plugin | What it does |
| --- | --- |
| [`domdomdom`](https://github.com/scruffymongrel/domdomdom) | Pipe JS via stdin to evaluate against any HTML page. Lightweight happy-dom alternative to Playwright. |
